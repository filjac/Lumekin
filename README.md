# LumeKin

An asynchronous digital board game for 2–6 players. Collect and develop creatures called Kins, battle in escalating arena formats across 9 laps, then compete in a playoff.

## Files

| File | Description |
|---|---|
| `lumekin-board.html` | Main game prototype |
| `lumekin-rulebook.html` | Complete rulebook |
| `kin-cards-demo.html` | Kin card designs |
| `card-backs-demo.html` | Card back designs |
| `element-logos.html` | Element logo preview |

## Game Overview

- **2–6 players** · **9 laps** · **60 squares** · **36 Kins** · **6 elements**
- Elements: 🔥 Fire · 💧 Water · ⚡ Electric · 🌱 Earth · 🌬️ Air · 🌑 Dark
- Arena format escalates: 1v1 → 5v5 → 1v1
- Playoff after lap 9: QF → SF → Grand Final
